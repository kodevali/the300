# **App Name**: TeamView

## Core Features:

- Data Card Display: Display data cards showing the count of employees using each OS, and total employees.
- Employee Table: Display employee data in a sortable and filterable table with columns for OS, Name, Email, Department, Line Manager, and Line of Business.
- Interactive Filtering: Filter the employee table based on the OS selected from the data cards.
- AI-Powered Line Manager Suggestion: AI-powered tool to analyze employee data and suggest potential line manager realignments to improve team efficiency based on LOB, skills and historical data.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F5F5F5), near white, for a clean, modern look.
- Accent color: Teal (#009688) to highlight interactive elements and data cards.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, readable style.
- Use official logos/icons for Google Chrome and Microsoft for OS column and interactive data cards. Use simple, professional icons for other columns.
- Data cards at the top, followed by the employee table below. Use a responsive layout for different screen sizes.
- Subtle transitions when filtering and sorting the table for a smooth user experience.
